Dark Mode Toggle App

This mini application demonstrates how the localStorage API works in JavaScript. The toggle button switches between light mode and dark mode.

When you change the theme, the selected theme is saved in localStorage. This means that even if you refresh the page or reopen the browser, the website will remember your selected theme.

Features of this Mini App

  • Toggle between Light Mode and Dark Mode
  • Theme preference saved using localStorage
  • Theme remains even after refreshing the page
  • Simple inline CSS styling
